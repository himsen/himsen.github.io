One-time-pad

Often abbreviated OTP. For a description of the one time pad encryption/decryption see the wiki page. There is a good reason it's called a ONE time pad; security will be lost if encrypting with the same key more than once. Some simple implementations of the OTP: Java, C++, C and Python. The following program can be used to attack an OTP encryption if the key has been used more than once. You may input as many texts as you want and actually the more texts you input the better the attack is because it removes ambiguity. Downlad.
